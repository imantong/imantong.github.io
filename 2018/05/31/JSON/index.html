JSON

发表于 2018-05-31

JSON(JavaScript Object Notation, JavaScript对象表示法)是一种数据格式，也是一个JS对象。

JSON语法

JSON可以表示的值：

  • 简单值：数值、布尔值、字符串、null（不支持undefined）
  • 对象
  • 数组

简单值

SON字符串必须使用双引号（单引号会导致语法错误）

对象

JSON中对象的属性必须加双引号。

JSON对象与JavaScript对象字面量的区别：

  1. 没有变量声明（JSON没有变量）
  2. 末尾没有分号（因为这不是JavaScript语句）
  3. 属性名必须加双引号

数组

JSON数组采用的就是JavaScript中的数组字面量形式，同JSON对象，JSON数组也没有变量和分号。

解析与序列化

ES5定义了全局对象JSON，JSON对象有两个方法：

JSON.stringify()：把JavaScript对象序列化为JSON字符串

JSON.parse()：把JSON字符串解析为JavaScript对象
