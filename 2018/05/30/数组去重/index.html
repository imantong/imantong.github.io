title: 数组去重
date: 2018-05-30
tags:

- JavaScript

unique([1,3,3,2]) //return [1,3,2]

unique([1,3,’3’,2]) //return [1,3,’3’,2]

1.普通版

时间复杂度为O(n^2)

定义一个变量数组res保存结果，遍历需要去重的数组，遍历res，如果该元素已经存在res中，则为重复元素，否则添加到res中。

2.简化版

时间复杂度还是O(n^2)

3.Hash结构

时间复杂度为O(n)

定义一个对象obj，将数组元素作为对象的键，遍历数组，如果键已经存在，则为重复元素，否则将元素存入res结果数组，并为obj添加键。

4.hasOwnProperty

通过对象的hasOwnProperty()方法检测给定的字符串是否是对象的自有属性。

5.区分对象类型

上面的typeof不能区分两个对象，比如{val:1}和{val:2}，typeof item + item都是”object[object Object]”，可以使用JSON.stringify将对象序列化。

6.ES6 Map

JavaScript的对象本质上是键值对的集合（Hash结构），但是只能用字符串作为键。
ES6的Map和对象类似，但“键”的范围不限于字符串，可以是各种类型的值。

7.ES6 Set

Set类似于数组，但是成员的值都是唯一的，没有重复的值。
